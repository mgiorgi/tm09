clean_column_value(v)

There are two basic ways to clean a column‘s value: h() and sanitize(). The latter is useful when the column contains valid html data, and you want to just disable any scripting. People can always use field overrides to clean data one way or the other, but having this override lets people decide which way it should happen by default.

Why is it not a configuration option? Because it seems like a somewhat rare request. But it could eventually be an option in config.list (and config.show, I guess).

 5:       def get_column_value(record, column)
 6:         # check for an override helper
 7:         value = if column_override? column
 8:           # we only pass the record as the argument. we previously also passed the formatted_value,
 9:           # but mike perham pointed out that prohibited the usage of overrides to improve on the
10:           # performance of our default formatting. see issue #138.
11:           send(column_override(column), record)
12:         # second, check if the dev has specified a valid list_ui for this column
13:         elsif column.list_ui and override_column_ui?(column.list_ui)
14:           send(override_column_ui(column.list_ui), column, record)
15: 
16:         elsif column.inplace_edit and record.authorized_for?(:action => :update, :column => column.name)
17:           active_scaffold_inplace_edit(record, column)
18:         elsif column.column and override_column_ui?(column.column.type)
19:           send(override_column_ui(column.column.type), column, record)
20:         else
21:           value = record.send(column.name)
22: 
23:           if column.association.nil? or column_empty?(value)
24:             formatted_value = clean_column_value(format_value(value, column.options))
25:           else
26:             case column.association.macro
27:               when :has_one, :belongs_to
28:                 formatted_value = clean_column_value(format_value(value.to_label))
29: 
30:               when :has_many, :has_and_belongs_to_many
31:                 if column.associated_limit.nil?
32:                   firsts = value.collect { |v| v.to_label }
33:                 else
34:                   firsts = value.first(column.associated_limit + 1).collect { |v| v.to_label }
35:                   firsts[column.associated_limit] = '…' if firsts.length > column.associated_limit
36:                 end
37:                 formatted_value = clean_column_value(format_value(firsts.join(', ')))
38:                 formatted_value << " (#{value.length})" if column.associated_number? and column.associated_limit and firsts.length > column.associated_limit
39:                 formatted_value
40:             end
41:           end
42: 
43:           formatted_value
44:         end
45: 
46:         value = '&nbsp;' if value.nil? or (value.respond_to?(:empty?) and value.empty?) # fix for IE 6
47:         return value
48:       end
