Module ActiveScaffold::Actions::Nested
In: vendor/plugins/active_scaffold/lib/active_scaffold/actions/nested.rb

The Nested module basically handles automatically linking controllers together. It does this by creating column links with the right parameters, and by providing any supporting systems (like a /:controller/nested action for returning associated scaffolds).

links_for_associations()

Create the automatic column links. Note that this has to happen when configuration is done, because otherwise the Nested module could be disabled. Actually, it could still be disabled later, couldn‘t it? TODO: This should really be a post-config routine, instead of a before_filter.

[Source]

    # File vendor/plugins/active_scaffold/lib/active_scaffold/actions/nested.rb, line 32
32:     def links_for_associations
33:       active_scaffold_config.list.columns.each do |column|
34:         # if column.link == false we won't create a link. that's how a dev can suppress the auto links.
35:         if column.association and column.link.nil?
36:           if column.plural_association?
37:             # note: we can't create nested scaffolds on :through associations because there's no reverse association.
38:             column.set_link('nested', :parameters => {:associations => column.name.to_sym}) #unless column.through_association?
39:           elsif not column.polymorphic_association?
40:             model = column.association.klass
41:             begin
42:               controller = self.class.active_scaffold_controller_for(model)
43:             rescue ActiveScaffold::ControllerNotFound
44:               next
45:             end
46: 
47:             actions = controller.active_scaffold_config.actions
48:             action = nil
49:             if actions.include? :update and column.actions_for_association_links.include? :edit and model.authorized_for? :action => :update
50:               action = 'edit'
51:             elsif actions.include? :show and column.actions_for_association_links.include? :show and model.authorized_for? :action => :read
52:               action = 'show'
53:             end
54:             column.set_link(action, :controller => controller.controller_path, :parameters => {:parent_controller => params[:controller]}) if action
55:           end
56:         end
57:       end
58:     end
